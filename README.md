# Purely Organic

An e-commerce web application designed for selling organic products online.

## Overview

Purely Organic is a full-stack web application that provides customers with a seamless shopping experience while enabling administrators to manage products and orders efficiently.

The project was built to strengthen my understanding of backend development, authentication, database management, and full-stack application architecture.

---

## Features

- User Authentication
- Product Catalog
- Shopping Cart
- Order Management
- Admin Dashboard
- Responsive Design
- Product Search
- Secure Session Handling
